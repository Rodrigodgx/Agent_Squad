# Implementação do Menu de Habilidades dos Lancers no Mapa

## Resumo da Tarefa

Esta tarefa visa implementar um menu de habilidades acessível diretamente no mapa ao interagir com os Lancers. O objetivo é otimizar o fluxo de jogo, permitindo que os jogadores planejem e executem suas estratégias de forma mais rápida e eficiente, sem a necessidade de navegar por múltiplos menus. A implementação deve incluir a exibição de habilidades únicas para cada Lancer, a possibilidade de clicar nessas habilidades para posicioná-las no mapa, e um sistema simples para substituir as imagens das habilidades sem necessidade de alterações complexas no código.

Critérios de Aceite: Ao passar o mouse sobre um Lancer no mapa, um menu expansível deve aparecer, exibindo 3 habilidades únicas. A interface deve ser intuitiva e o desempenho do jogo não deve ser impactado.
